# Wi-Fi Range Extender using Raspberry Pi

## Aim
To develop a Wi-Fi Range Extender using Raspberry Pi that improves wireless coverage in a specified area, eliminates Wi-Fi dead zones, and enhances connectivity while remaining cost-effective and compatible with standard IEEE 802.11 Wi-Fi protocols.

GitHub Repository Link:
https://github.com/yourusername/wifi-range-extender-raspberrypi

---

## Problem Statement
Wireless networks often experience weak signal areas or dead zones due to obstacles, distance from the router, or signal interference. These dead zones result in poor connectivity and reduced internet performance. This project proposes a Raspberry Pi based Wi-Fi range extender to rebroadcast the existing Wi-Fi signal and extend network coverage.

---

## Scope of the Solution
- Extend wireless network coverage.
- Reduce Wi-Fi dead zones.
- Improve connectivity for multiple devices.
- Provide a low-cost alternative to commercial Wi-Fi extenders.
- Compatible with IEEE 802.11 Wi-Fi standards.

Future Scope:
- Web-based configuration interface
- Automatic channel optimization
- Network monitoring dashboard
- Mesh networking support

---

## Software / IDE Used
- Raspberry Pi OS
- Cisco Packet Tracer (for simulation)

---

## System Architecture

Internet → Main Router → Raspberry Pi Extender → Client Devices

The Raspberry Pi receives the Wi-Fi signal from the main router and rebroadcasts it to extend the network coverage.

---

## Conclusion
This project demonstrates a cost-effective method to extend wireless network coverage using Raspberry Pi. The system successfully improves connectivity in areas with weak Wi-Fi signals and provides an affordable alternative to commercial Wi-Fi extenders.
